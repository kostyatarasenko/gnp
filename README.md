### Installation
1. Clone the repo
   ```sh
   git clone git@github.com:kostyatarasenko/gnp.git
    ```
2. Install python v2.7 or higher (recommended 2.7)
3. Install cryptography package
   ```sh
   pip install cryptography
   ```
4. Run the installation script (example for zsh)
   ```sh
   sudo zsh install.sh
   ```
5. Enjoy!
